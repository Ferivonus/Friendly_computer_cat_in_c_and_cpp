# Friendly_computer_cat

## vcpkg dependencies:

- raylib
- nlohmann-json
- sqlite3
- cpp-httplib
- spdlog
- pthread

Create a vcpkg_installed directory in the root of the project and copy the installed libraries from vcpkg to that directory. This is necessary to ensure that the project can find the required libraries when building and running.

I am not sharing my builtin-baseline public.
